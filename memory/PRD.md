# PlastiLu Ind. Embalagens - Bio Link PRD

## Problema Original
Criar um site exclusivo para link na bio do Instagram da empresa PlastiLu Ind. Embalagens, substituindo o Linktree/Bio.site por uma página própria com design extremamente premium, sofisticado e corporativo de alto padrão.

## User Personas
- **Clientes B2B**: Empresas industriais buscando fornecedores de embalagens plásticas
- **Novos leads**: Visitantes do Instagram querendo conhecer a empresa
- **Clientes atuais**: Precisam de acesso rápido a WhatsApp, catálogo e localização

## Requisitos Core (Estáticos)
- Design premium corporativo industrial
- Paleta: Azul luxo profundo + Preto com degradê
- Mobile-first otimizado para Instagram bio
- Carregamento rápido
- 5 botões principais de CTA
- Animações sutis e elegantes
